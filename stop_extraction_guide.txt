========================================
停止提取功能使用说明
========================================

新增功能
--------
现在你可以在大模型提取过程中随时停止提取!

界面变化
--------
在"🤖 大模型提取"区域,按钮布局变为:

[启动提取] [停止提取] [查看结果]

- 启动提取: 橙色按钮,开始提取时禁用
- 停止提取: 红色按钮,提取过程中启用
- 查看结果: 提取完成后启用


使用方法
--------
1. 点击"启动提取"开始大模型提取
2. 提取过程中,"停止提取"按钮变为可用(红色)
3. 如需停止,点击"停止提取"按钮
4. 系统会弹出确认对话框
5. 确认后,提取会在当前任务完成后停止


停止时机说明
------------
提取过程是分批次进行的:
- 文本提取: 按文本块批次(默认5个一批)
- 图像提取: 按图像批次(默认2个一批)

点击停止后:
✓ 当前正在处理的API请求会完成(最多120秒)
✓ 后续批次的任务会被取消
✓ 已处理的数据不会保存
✓ 系统会清理状态,可以重新开始


日志输出示例
------------
正常启动:
  [提取] 使用自定义输出目录: D:\data\results
  [提取] ===== 大模型提取流程开始 =====
  [INFO] 阶段 1: LLM 文本提取开始

点击停止:
  [提取] 用户请求停止提取...
  [提取] ===== 大模型提取已停止 =====
  [提取] 用户手动停止提取,结果未保存


注意事项
--------
⚠ 停止后不会保存任何结果
⚠ 已消耗的API调用不会退款
⚠ 当前正在进行的API请求会完成(最多120秒)
⚠ 停止后可以立即重新开始提取


技术实现
--------
- 停止标志: extract_stop_flag
- 检查点: 每个进度回调时检查
- 超时控制: API调用120秒超时
- 清理机制: 自动重置UI状态
